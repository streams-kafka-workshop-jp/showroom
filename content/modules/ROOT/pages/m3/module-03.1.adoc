== おすすめ製品
前のモジュールでは、Globex Webアプリケーションを使用していくつかのユーザーアクティビティイベントを生成しました。それでは、そのアクティビティに基づいて、おすすめ製品のリストがリアルタイムでどのように計算されるかを見てみましょう。



* {streams} https://amq-streams-console-{user_name}.{openshift_cluster_subdomain}/kafka[console, window="_amqstreams"] のtopicリストに戻ると（左側の *Topics* をクリック）、*globex.recommendation* で始まる複数のtopicがあることに気付くでしょう。これらは、Kafka Streams APIを使用してユーザーアクティビティストリームに基づいて最も人気のある製品のリストを計算する_recommendation engine_アプリケーションによって作成されたtopicです。
+
image::m3/globex-recommendation-topics-console.png[]

* *globex.recommendation-product-score-aggregated-changelog* topicには、最も人気のある製品のリストを含むメッセージが格納されており、最新のメッセージに現在のリストが含まれています。
+
image::m3/amqstreams-globex-tracking-recommendation.png[]

* Globex Webアプリケーションはこのtopicのメッセージを使用して、ホームページにおすすめ製品のリストを表示します。

* これを確認するには、ブラウザからlink:{globex_web_url}[Globex web application, window="_globex"]を指すタブに移動します。ホームページに移動すると、以前に「いいね」した製品のリストに対応する、おすすめ製品のリストにいくつかのアイテムが表示されるはずです。
+
image::m3/globex-home-page-featured.png[]

* おすすめ製品のリストがリアルタイムで計算されることを実証するために、Globex Webアプリケーションの隣にデプロイされたシミュレータを使用して、多数のユーザーアクティビティをシミュレートできます。


* 新しいブラウザタブでシミュレータを開くにはクリックしてください - https://activity-tracking-simulator-globex-{user_name}.{openshift_cluster_subdomain}/q/swagger-ui/[Activity Tracking Simulator, window="_simulator"]


* これにより、シミュレータのREST APIを表示するSwagger UIページが開きます。
+
image::m3/swagger-ui-activity-tracking-simulator.png[]

* *POST* リンクをクリックし、次に右側の *Try it out* リンクをクリックします。+
+
image::m3/swagger-ui-activity-tracking-simulator-post-try.png[]

* 必要に応じて、生成されるユーザーアクティビティの数を変更できます（デフォルトは100）。 *Execute* をクリックして、シミュレータへのRESTコールを実行します。
+
image::m3/swagger-ui-activity-tracking-simulator-2.png[]

* ブラウザウィンドウで、https://amq-streams-console-{user_name}.{openshift_cluster_subdomain}/kafka[streams for Apache Kafka console, window="_amqstreams"]のタブに移動し、*globex.tracking* topicでメッセージが生成される様子を確認します。

* 数秒後、新しく計算されたおすすめ製品のリストを反映した新しいメッセージが *globex.recommendation-product-score-aggregated-changelog* topicにも表示されるはずです。

* link:{globex_web_url}[Globex web application, window="_globex"]のブラウザタブに移動し、ホームページを更新して、継続的に再計算されているおすすめ製品のリストがどのように変化したかを確認してください。

素晴らしい！おすすめ製品のリストは、ユーザーアクティビティストリームに基づいてリアルタイムで更新されます。しかし、リストがどのように計算されるか、そしてKafka Streamsがどのようにして舞台裏で魔法のようにリストを生成するかを詳しく見てみましょう。
